## Template de démo
Stack : HTML + CSS + JS vanilla
- Chatbot widget simulé (sans vraie IA — réponses pré-scriptées)
- Design aux couleurs du client
- Déployé automatiquement via Vercel depuis GitHub
- Livraison en <1 jour ouvré

## Leçons apprises
- Les démos HTML convertissent mieux qu'un pitch deck
- Le prospect peut "tester" son futur bot avant de signer
- Toujours personnaliser les couleurs et le logo du client
- Prévoir 3-5 scénarios de conversation pré-scriptés représentatifs du vrai bot
- Une fois la démo validée → développer le vrai bot avec la vraie stack H'appi
