DropOS is a web-first SaaS platform for dropshipping optimization.

**Path**: `/home/v-nbayonne/DropOS/`
**Stack**: Next.js (frontend) + FastAPI + PostgreSQL (same backend pattern as Quality Tracking App)
**Positioning**: "The operating system for your dropshipping business" — one subscription replacing $400–$1,000/mo of tools

**Target user**: Dropshippers doing $10K–$200K/month frustrated by tool sprawl (AutoDS + Klaviyo + TrueProfit + AfterShip + Gorgias + Loox...)

**Pricing**: $99–$149/mo flat (vs $400–$1,000/mo current stack)

**Roadmap**:
- Phase 1 MVP: Analytics core — true net profit tracker, landed cost calculator (post-tariff), multi-store dashboard
- Phase 2: Fulfillment — multi-supplier sync, inventory failover, returns portal
- Phase 3: Sourcing — product research, supplier scoring, trend finder
- Phase 4: Marketing — email flows, review automation, ad tracking

**Key differentiators**:
1. True net profit (captures chargebacks, duties, refund fees, currency losses — not just COGS + ad spend)
2. Landed cost engine with live US HTS / EU TARIC tariff data (critical post May 2025 de minimis closure)
3. Cross-module data intelligence (e.g. high-return-rate products auto-suppressed from marketing)
4. Shopify Partner API as primary integration channel for MVP distribution

**Why:** No affordable tool ($500/mo) fills this gap at SMB scale. AutoDS is hated, Zendrop overcharges, BeProfit has broken ad tracking. Trust gap is wide open.
